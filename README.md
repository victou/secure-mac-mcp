# secure-mac-mcp

Secure Mac MCP is an early-stage project for building tools around macOS security workflows through the Model Context Protocol.

The implementation stack is intentionally not selected yet. This repository currently contains the project baseline and agent instructions so future work can start from a clean, shared context.

## Getting Started

```sh
git clone git@github.com:victou/secure-mac-mcp.git
cd secure-mac-mcp
```
